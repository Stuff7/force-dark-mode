# Changelog

---

## [3.1.0] - 2025-11-30
### Added
- **Per-Site Configurations:** Customize filter settings for individual sites via the ⚙️ gear icon. Presets are available for quicker setup, allowing users to adjust darkness and other filter effects to their preference.  
- **Additional Filters:** Beyond invert and hue rotation, new filters have been added to enhance dark mode appearance and flexibility.  

### Changed
- **Zap Mode Improvements:**  
  - Duplicate selectors are now blocked to prevent redundancy.  
  - Hovering over already added selectors highlights them in red for better visibility.  
  - Selectors can now be removed directly from Zap Mode for convenience.  

### Removed
- Blacklist selector removal animation has been removed for a cleaner, faster experience.  

---

## [3.0.1] - 2025-08-31
### Changed
- Increased Zap Mode delete transition speed.

---

## [3.0.0] - 2025-08-29
### Major Update
- **Configurable Dark Mode:** Fully configurable, fine-tune how and where it applies.  
- **Zap Mode ⚡:** Select elements on a page that should not be affected by Dark Mode.  
- **Customizable Keyboard Shortcut:** Configure your own shortcut to toggle Dark Mode instantly.  
- **Expanded Protocol Support:** Works across all protocols, not just http:// and https://.  

---

## [2.1.0] - 2025-07-03
### Changed
- Removed keyboard shortcut; enable/disable the extension by clicking the extension button.  
- Instead of forcing a dark background and light text, colors are now inverted for the whole site (except media) for more faithful visuals.  
- Manual site management: go to "Manage Extension > Preferences" to add/remove sites via a text area without visiting the site.
